**Goal:** Extrair, fiéis ao livro e validadas por visão, as criaturas (inéditas) + raças jogáveis + itens dos grupos "Áreas de Tormenta" (impressas 18–29) e "Capangas & Bandoleiros" (impressas 42–51) de Ameaças de Arton, reusando a fundação multi-fonte e o pipeline do spike.

**Architecture:** Sem código novo — só dados em `data/ameacas-de-arton/` + artes em `site/public/`. Pipeline: render 300 DPI + `pdftotext -layout` → extração por visão (passada 1) com **reprint-check contra o Básico** → revisor independente por visão (passada 2, meta 0 discrepâncias) → raças cruzadas → artes. Build estático valida tudo; a guarda `idsDuplicados` barra reprints que escaparem.

**Tech Stack:** poppler em `extracao/poppler-bin/poppler-26.02.0/Library/bin`; Node em `/c/Program Files/nodejs` (Bash: `export PATH="$PATH:/c/Program Files/nodejs"`). Offset de Ameaças: **PDF = impressa + 2**.

**Convenções:** comandos de site em `site/`, de extração em `extracao/`; `git add <paths>` específicos, nunca `-A`; sem push; caminhos com espaço entre aspas. Branch atual: `fase2-ameacas-spike` (a Onda 1 continua nela).

**Formato de `criatura` (exemplo real, `data/livro-basico/criaturas/bandido.json`):**
```json
{"id":"bandido","tipo":"criatura","nome":"Bandido","resumo":"...","fonte":{"livro":"livro-basico","pagina":289},"imagens":[],"secoes":[{"titulo":"","texto":"..."}],"relacoes":[],"mecanica":{"nd":"1/4","tipo":"Humanoide (humano)","tamanho":"Médio","tema":"Ermos","iniciativa":"+4","percepcao":"+1","defesa":"13","fortitude":"+1","reflexos":"+3","vontade":"–1","pontosDeVida":"6","deslocamento":"9m (6q)","ataques":["Corpo a Corpo Clava +7 (1d6+3)."],"atributos":{"forca":"1","destreza":"2","constituicao":"1","inteligencia":"0","sabedoria":"–1","carisma":"0"},"pericias":"Furtividade +5.","equipamento":"Clava.","tesouro":"Metade.","habilidades":[]}}
```
Regras: todos os valores são strings; negativos com en-dash `–`; `atributos` = `forca,destreza,constituicao,inteligencia,sabedoria,carisma`; `ataques` = só linhas Corpo a Corpo/À Distância (Sopro/auras/magias vão em `habilidades:[{nome,descricao}]`); palavras defensivas finais ficam no `defesa` (ex.: `"22, redução de dano 10"`); `mecanica.tema` = nome do grupo; `fonte.pagina` = impressa real; `pontosDeMana` só se houver.

**Formato de `raca` (exemplo real, `data/livro-basico/criaturas/...` → ver `data/livro-basico/racas/golem.json`):** `mecanica = { modificadores:[{atributo,valor:NUMBER}], tamanho, deslocamento:NUMBER, deslocamentoUnidade:"m", nota?, habilidades:[{nome,descricao,efeito?}] }`. Modificador `valor` é número (`-1`, não en-dash). Para "+1 em atributo à escolha", usar `{valor:1,escolha:true}` (sem `atributo`).

## Task 1: Render + texto dos dois grupos (cache)

**Files:** Create (gitignored): `extracao/cache/ameacas/areas-tormenta/`, `extracao/cache/ameacas/capangas/`

- [ ] **Step 1: Renderizar Áreas de Tormenta (impressas 18–29 = PDF 20–31) e Capangas (impressas 42–51 = PDF 44–53) em 300 DPI + texto.**

Run (de `extracao/`):
```bash
export PATH="$PATH:/c/Program Files/nodejs"
bin="poppler-bin/poppler-26.02.0/Library/bin"
pdf="../pdfs/Ameacas-de-Arton-v1.0-17-11-2023.pdf"
mkdir -p cache/ameacas/areas-tormenta cache/ameacas/capangas
for p in $(seq 20 31); do "$bin/pdftoppm.exe" -f $p -l $p -png -r 300 "$pdf" "cache/ameacas/areas-tormenta/pdf-$p"; "$bin/pdftotext.exe" -f $p -l $p -layout -enc UTF-8 "$pdf" "cache/ameacas/areas-tormenta/txt-impressa-$((p-2))_pdf-$p.txt"; done
for p in $(seq 44 53); do "$bin/pdftoppm.exe" -f $p -l $p -png -r 300 "$pdf" "cache/ameacas/capangas/pdf-$p"; "$bin/pdftotext.exe" -f $p -l $p -layout -enc UTF-8 "$pdf" "cache/ameacas/capangas/txt-impressa-$((p-2))_pdf-$p.txt"; done
ls cache/ameacas/areas-tormenta/ cache/ameacas/capangas/
```

Se divergir, ajustar o range e re-renderizar. Confirmar acentos no `.txt` (ex.: "Tormenta", "Hyninn").
